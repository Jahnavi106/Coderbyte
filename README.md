# Coderbyte
A simple StringChallenge by [Coderbyte](https://coderbyte.com/challenges#)
Challenge is: if the number N has N consecutive numbers whose value is again N then the code should output "true" .. else "false".

Examples: 
- if 
  - **input**="567432223466"
  - **output**="true"

- if 
  - **input**="456893454"
  - **output**="false"
